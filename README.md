A data analytics project simulating real-world mining operations across 
Western Australia — covering production output, equipment performance, 
and safety incident tracking.

## 📊 Live Dashboard
👉 [Click here to view the dashboard](https://datastudio.google.com/reporting/f98ca8ed-99b3-48e5-97c9-fcb9d0b1a5e0)

## 🧰 Tools Used
- Python (pandas, numpy) — data generation & cleaning
- Google Colab — coding environment
- Google Looker Studio — dashboard & visualization

## 📁 Dataset
- **Source:** Synthetic dataset (Python generated)
- **Industry:** Mining & Resources — Western Australia
- **Sites Covered:** Olympic Dam, Pilbara Iron, Hunter Coal,
  Kalgoorlie Gold, Greenbushes Lithium

## 🧹 What I Cleaned
- Removed duplicate records
- Fixed missing operational values
- Corrected data types (dates, percentages, tonnage)
- Standardized site names and column headers
- Engineered safety and uptime performance metrics

## 📈 Dashboard Insights
- ⚠️ Total Safety Incidents: 99
- ⚙️ Equipment Uptime: 92.1%
- 🏆 Highest Production Site: Olympic Dam (11.4M tons)
- 🪨 Top Commodities: Iron, Coal, Gold, Lithium
- 📉 Production tracked across all 5 major WA mine sites
